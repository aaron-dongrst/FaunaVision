# Training Scripts

## Quick Start

```bash
./scripts/run_all_steps.sh
```

This runs everything automatically.

## Individual Scripts

1. **`extract_frames.py`** - Extract frames from videos
2. **`prepare_yolo_dataset.py`** - Prepare YOLO dataset
3. **`train_pig_behavior.py`** - Train model
4. **`test_pig_model.py`** - Test model
