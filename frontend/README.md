# Frontend - PigVision

React application for pig behavior analysis.

## Setup

```bash
npm install
npm start
```

Opens at `http://localhost:3000`

## Features

- Upload pig videos
- Enter pig information
- View behavior analysis
- See distress assessment

## Components

- `App.js` - Main app
- `Animal.js` - Pig card
- `AnimalParameters.js` - Pig info form
- `VideoUpload.js` - Video upload
- `AnalysisResults.js` - Results display
